## 参与开发

```
git clone https://github.com/GalAster/Soramimi.git --depth=1
```

`git` 只包含纯文本的最终作品, 不包含视频构建工程和素材.

## 目录结构

```yaml
- Tools      (构建脚本)
- Creations  (作品主目录)
    - Fonts  (字幕字体)
    - Series (作品系列)
    - Single (单曲创作)
    - Theme  (主题系列)
```

## 阶段标记

- 🎬(Stage 1): 计划阶段
- 🔥(Stage 2): 准备状态
- 🚀(Stage 3): 创作阶段
- 🎼(Stage 4): 完结状态
- 🎨(Stage 5): 修订阶段
